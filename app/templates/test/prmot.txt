Ты — эксперт frontend-разработки уровня senior. Создай полностью готовый, красивый и современный frontend для сайта «ТарифМатч» — сервис подбора лучших мобильных тарифов под пользователя.

Сайт состоит ровно из 4 отдельных HTML-файлов. Каждый файл должен быть полностью самодостаточным: весь CSS внутри <style>, весь JS внутри <script>, без внешних файлов. Используй Tailwind CSS через CDN[](https://cdn.tailwindcss.com) + любые нужные CDN (Chart.js для диаграмм, Font Awesome для иконок). Дизайн должен быть премиальным: тёмно-синяя/градиентная тема (цвета операторов), плавные анимации, glassmorphism-эффекты, адаптивность (мобильные + десктоп), карточки с hover-эффектами.

Страницы:

1. index.html — главная (landing)
   - Красивый hero с заголовком «Найди идеальный мобильный тариф за 30 секунд»
   - Короткое описание сервиса
   - 3 большие кнопки: «Обычный поиск», «AI-подбор», «Сравнить тарифы»
   - Футер

2. search.html — обычный поиск с мощными фильтрами
   Фильтры (все обновляют результаты в реальном времени):
   - Бюджет: двойной range-слайдер (два ползунка) от 0 до 3000 ₽ с подписями «от» и «до». Мин не может быть больше макс и наоборот.
   - Интернет: двойной range-слайдер 0–50 ГБ. Если значение ≥ 50 — автоматически «Безлимит» + красивая надпись.
   - Минуты: двойной range-слайдер 0–2000. Если ≥ 2000 — «Безлимит».
   - SMS: двойной range-слайдер 0–1200. Если ≥ 1200 — «Безлимит».
   - Чекбоксы «Безлимит на»: Мессенджеры, Соцсети, Видеохостинги, Музыка (с иконками).
   - Операторы (мульти-выбор с логотипами/цветами):
     • МТС (красный M)
     • Билайн (жёлтый B)
     • Tele2 (чёрный T2)
     • Мегафон (зелёный M)
     • Yota (синий Y)

   Результаты: сетка карточек тарифов (название, оператор, цена в месяц, основные параметры, кнопки «Подробнее» и «Добавить в сравнение»).
   Жёстко зашитый массив из 12–15 реальных тарифов (придумай правдоподобные данные). JS-фильтрация + сортировка (по цене, по интернету).

3. compare.html — сравнение двух тарифов
   - Два выбранных тарифа
   - Две колонки с детальной информацией
   - Большая radar-диаграмма (Chart.js) — 5 осей: Цена, Интернет, Минуты, SMS, Доп.безлимиты
   - Таблица сравнения параметров
   - Кнопки «Заменить тариф», «Вернуться в поиск», «Поделиться сравнением»

4. search_ai.html — AI-подбор
   - Красивый интерфейс «Расскажи, как ты пользуешься телефоном»
   - Несколько быстрых вопросов + большое поле «Опиши своими словами»
   - Кнопка «Найти идеальный тариф с ИИ»
   - После нажатия — красивая анимация загрузки и выдача топ-3 тарифов с объяснением «почему именно эти» (симуляция AI через JS)

Требования к коду:
- Современный, чистый, комментарии на русском
- Все слайдеры работают идеально (реализуй двойные range-слайдеры через два input[type=range] + JS-синхронизация)
- При выборе тарифа для сравнения
- Навигация между страницами через <a href> (всё в одной папке)
- Мобильная версия идеальная (Tailwind responsive)
- Анимации (framer-like hover, fade-in результатов)
- Никакого внешнего backend — всё на mock-данных

Выведи код каждого файла полностью и отдельно, начиная с --- index.html --- и заканчивая закрывающим тегом </html>. Сначала index.html, потом search.html, compare.html и search_ai.html.
